# Krigacy

> Krigacy = Kritor Legacy

Kritor 的 Onebot 适配器，用于将 Kritor 标准转换为 Onebot/go-cqhttp 标准

本项目的定位为 Kritor 应用端，Onebot 协议端。

# 构建

拉取仓库或更新 Kritor 后，需要执行一次
```shell
./gradlew generateProto
```
